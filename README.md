# Odisseus Raspberry Pi

## Hardware

- 4 Wheels
- 4 DC motors
- A Pi camera
- An ultrasound sensor
- An IR sensor
- A DC motor hat for raspberry pi

## Installation

Make sure that the raspberry pi used satisfies the  requirements in _requirements.txt_
